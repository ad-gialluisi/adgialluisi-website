University

From 2013 to 2025, I attended the bachelor graduation course (the so-called “Laurea triennale”) in Informatica e Tecnologie per la produzione del Software (Computer science and software producing technologies), at the University of Bari “Aldo Moro”.

Here follow some of the projects/materials I created/read to pass exams.
This non-exaustive list is given as it is, without any warranty; Hoping it will be considered useful to future computer science students.

  • Some materials for the “Algorithms and Data structures” exam.
    A set of articles and videos I found useful;

  • Some materials for the “Formal languages” exam.
    A set of links I found useful;

  • Some materials for the “Databases” exam.
    A set of materials I found useful, and three documents I wrote personally;

  • “Web programming” project: Project “Area informatica”.
    A website that should represent an e-learning, made in the far 2015, using PHP/HTML.
    It is available at the git repository https://github.com/ad-gialluisi/uni-webdev;

  • “Software engineering” project: Project “Cicerone”.
    A websute related to low-cost and unusual trips, developed in PHP/HTML/CSS/Javascript in the far 2019/2020.
    It is available at the git repository https://github.com/ad-gialluisi/uni-sweng;

Italiano

Università

Dal 2013 al 2025, ho frequentato il corso di Laurea triennale in Informatica e Tecnologie per la produzione del Software, all’Università degli studi di Bari “Aldo Moro”.

Qui seguono alcuni dei progetti/materiali che ho creato/letto per superare gli esami.
Non trattasi di una lista esaustiva, l’elenco viene fornito così com’è; Senza alcuna garanzia; Nella speranza che possa tornare utile ad eventuali futuri studenti di informatica.

  • Alcuni materiali per l’esame di “Algoritmi e strutture dati”.
    Un insieme di articoli e video che ho trovato utili;

  • Alcuni materiali per l’esame di “Linguaggi formali”.
    Un insieme di link che ho trovato utili;

  • Alcuni materiali per l’esame di “Basi di dati”.
    Un insieme di materiali che ho trovato utili, nonchè, tre documenti che ho scritto personalmente;

  • Progetto di “Programmazione per il web”: Progetto “Area informatica”.
    Un sito web che dovrebbe rappresentare un e-learning, creato nel lontano 2015, sfruttando il solo PHP/HTML.
    Esso è disponibile al repository git https://github.com/ad-gialluisi/uni-webdev;

  • Progetto di “Ingegneria del software”: Progetto “Cicerone”.
    Un sito web inerente i viaggi low-cost e “non usuali”, sviluppato in PHP/HTML/CSS/Javascript nel lontano 2019/2020.
    Esso è disponibile al repository git https://github.com/ad-gialluisi/uni-sweng;
